# Roadmap

This roadmap outlines the planned evolution of Steam Media Pack.

It represents the project's current direction and may change as development progresses.

---

# Version 11.1 — Foundation

Focus on establishing a solid and maintainable foundation.

## Goals

- Complete project documentation
- Standardize repository structure
- Improve Steam integration
- Standardize project scripts
- Improve library artwork management
- Establish development standards

---

# Version 11.2 — Experience

Focus on improving usability and developer experience.

## Goals

- GitHub Pages documentation
- Technical diagrams
- Improved installation experience
- Better diagnostics
- Enhanced browser detection
- Documentation refinements

---

# Version 12.0 — Product

Focus on transforming Steam Media Pack into a complete product.

## Goals

- Graphical installer
- Plugin system
- YAML configuration
- Automatic updates
- Flatpak support
- AppImage support

---

# Long-Term Vision

Future development may include:

- Additional streaming services
- Additional browser support
- Improved launcher management
- Cross-platform enhancements
- Community contributions
