# Glossary

This glossary defines the main terms used throughout the Steam Media Pack documentation.

---

## App Mode

Launch mode that opens a streaming service in a standalone browser window with minimal browser interface.

---

## Browser Integration

The component responsible for detecting and launching supported web browsers.

---

## Developer Guide

Technical documentation intended for contributors and maintainers.

---

## Feature Branch

A Git branch created to implement a single feature, improvement, or documentation update.

---

## Homolog

The branch used to validate changes before they are promoted to production.

---

## Launch Mode

The method used to open a streaming service.

Supported modes include:

- Auto Mode
- App Mode
- Kiosk Mode

---

## Launcher

A script responsible for starting a specific streaming service.

---

## Library Artwork

Images displayed by Steam for non-Steam shortcuts.

Supported artwork types include:

- Grid
- Hero
- Logo
- Icon

---

## Maintenance Tools

Scripts responsible for installation, updates, backup, restore, diagnostics, and other maintenance tasks.

---

## Non-Steam Shortcut

A shortcut manually or programmatically added to the Steam Library.

---

## Steam Integration

The process of creating and maintaining Steam shortcuts for supported streaming services.

---

## User Guide

Documentation intended for end users.
