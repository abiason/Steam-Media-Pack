# Configuration

This guide explains the available configuration options for Steam Media Pack.

Most settings are configured automatically during installation. In most cases, no manual intervention is required.

---

# Browser Support

Steam Media Pack supports Chromium-based browsers.

Supported browsers:

- Chromium
- Google Chrome
- Microsoft Edge
- Brave

Firefox support is planned for a future release.

---

# Display Detection

Steam Media Pack automatically detects your display resolution whenever possible.

This allows launchers to adapt to different devices without additional configuration.

---
