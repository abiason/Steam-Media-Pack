# Screenshots

This document defines the standards for screenshots used by Steam Media Pack.

Consistent screenshots improve documentation quality and visual presentation.

---

# General Guidelines

Screenshots should:

- represent the latest version of the project;
- be captured at native resolution;
- be sharp and uncompressed;
- avoid unnecessary desktop elements;
- focus on the relevant content.

---

# Resolution

Whenever possible, capture screenshots at:

- 1920×1080 (Full HD)
- 2560×1440 (QHD)
- Native application resolution

---

# Environment

Use a clean environment:

- Default wallpaper or neutral background
- No unrelated applications
- No personal information
- No notifications

---

# Steam Library

When capturing Steam:

- Use the official artwork.
- Ensure artwork is fully loaded.
- Display the complete shortcut information.
- Avoid cropped interface elements.

---

# Browser

When capturing browser windows:

- Hide unnecessary toolbars when possible.
- Use the project's recommended launch mode.
- Avoid showing personal accounts or sensitive information.

---

# File Format

Preferred format:

- PNG

Avoid:

- JPEG with excessive compression
- Screenshots containing visible artifacts
