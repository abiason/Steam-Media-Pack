# Code Font

Use a monospace font for:

- Code snippets
- Commands
- File paths
- Configuration examples

Recommended:

- JetBrains Mono
- Fira Code
- Cascadia Code

# Text Hierarchy

| Element | Usage |
|----------|-------|
| H1 | Page title |
| H2 | Major section |
| H3 | Subsection |
| Body | Standard text |
| Code | Commands and source code |

---

# Guidelines

- Use consistent font sizes.
- Maintain adequate spacing.
- Avoid excessive font variations.
- Do not mix multiple font families unnecessarily.

---

# Accessibility

Typography should prioritize readability.

- Use sufficient font size.
- Maintain proper line spacing.
- Ensure adequate color contrast.
