# Logo

This document defines the official Steam Media Pack logo and its correct usage.

---

# Official Versions

The project provides the following logo variations:

- Full Color
- Light
- Dark
- Monochrome

Always use the version that provides the best contrast with the background.

---

# Clear Space

Maintain sufficient clear space around the logo.

No text, icons, or other visual elements should overlap or touch the logo.

---

# Minimum Size

The logo should remain readable at all sizes.

Do not use the logo below the minimum readable size.

---

# Backgrounds

The logo may be used on:

- Light backgrounds
- Dark backgrounds
- Transparent backgrounds

Always ensure adequate contrast.

---

# Incorrect Usage

Do not:

- stretch or compress the logo;
- rotate the logo;
- change its colors;
- apply shadows or effects;
- modify individual elements;
- place the logo on low-contrast backgrounds.

---

# File Formats

Official logo files are available in:

- SVG
- PNG
