# Templates

This document defines the official templates used by Steam Media Pack.

Templates ensure visual consistency across documentation, GitHub, presentations, and promotional materials.

---

# Purpose

Official templates provide:

- Consistent branding
- Standardized layouts
- Faster content creation
- Professional presentation

---

# Supported Templates

The project may include templates for:

- GitHub repository
- Documentation
- Social media
- Presentations
- Banners
- Feature announcements
- Release notes

---

# Design Principles

Templates should:

- follow the official color palette;
- use the official typography;
- use the official icons;
- maintain consistent spacing;
- prioritize readability.

---

# Editable Areas

Templates should clearly separate:

- Fixed branding elements
- Editable content
- Images
- Titles
- Supporting text

---

# File Formats

Editable templates should be provided in formats suitable for future updates.

Examples:

- SVG
- PPTX
- Draw.io
- Figma (optional)

Exported assets should be provided in:

- PNG
- PDF (when applicable)

---

# Asset Location

```text
docs/assets/templates/
```

---

# Naming Convention

Use descriptive file names.

Examples:

```text
github-banner.png
feature-announcement.pptx
release-template.svg
social-post-square.png
presentation-title.pptx
```

---

# Guidelines

- Always start from an official template.
- Do not modify branding elements.
- Keep layouts clean and consistent.
- Use official colors and typography.
- Ensure exported assets are high quality.
