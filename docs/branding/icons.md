# Icons

This document defines the icon style used by Steam Media Pack.

Icons should be simple, consistent, and easy to recognize.

---

# Style

Icons should be:

- Minimal
- Modern
- Flat
- Consistent
- Readable at small sizes

---

# Usage

Icons may be used in:

- Documentation
- User interface
- GitHub
- Website
- Social media
- Presentations

---

# Guidelines

- Use a consistent icon family.
- Maintain uniform stroke width.
- Keep icon proportions consistent.
- Avoid mixing different icon styles.

---

# File Formats

Official icons should be provided in:

- SVG
- PNG

---

# Asset Location

```text
docs/assets/branding/icons/
```

---

# Recommendations

Preferred icon libraries include:

- Material Symbols
- Tabler Icons
- Lucide
- Heroicons

Use a single icon library whenever possible to maintain visual consistency.
