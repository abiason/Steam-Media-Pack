# Steam Media Pack Brand Guidelines

> Official visual identity guide for the Steam Media Pack project.

---

# 1. Purpose

This document defines the official visual identity of Steam Media Pack and establishes guidelines for the correct use of the project's visual assets.

---

# 2. Brand Values

Steam Media Pack is built around five principles:

- Simplicity
- Automation
- Consistency
- Open Source
- Linux First

---

# 3. Color Palette

| Name | HEX | Usage |
|------|------|------|
| Steam Midnight | `#171A21` | Main background |
| Steam Blue | `#1B2838` | Panels |
| Tech Dual Cyan | `#66C0F4` | Highlights |
| Success Green | `#5BA32B` | Success states |
| Warning Orange | `#E8A317` | Warnings |
| White | `#FFFFFF` | Primary text |
| Gray | `#C7D5E0` | Secondary text |

---

# 4. Typography

| Usage | Font |
|------|------|
| Logo | Orbitron Bold |
| Headings | Orbitron SemiBold |
| Body | Inter |

---

# 5. Logo

Official files:

- logo.png
- logo-dark.png
- logo-light.png
- favicon.png

Rules:

- Do not distort the logo.
- Do not change its colors.
- Do not rotate the logo.
- Maintain clear space around it.

# 9. Screenshots

Screenshots should:

- use Steam Dark Theme;
- avoid personal information;
- be captured at native resolution;
- show the application in a clean state.

---

# 10. Diagrams

All diagrams must:

- use the official color palette;
- use consistent typography;
- maintain visual consistency.
