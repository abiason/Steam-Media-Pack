# Roadmap

A versão 11.0 é considerada a base estável atual.

## Ideias para versões futuras

- Interface gráfica para seleção dos serviços.
- Instalação seletiva por serviço.
- Cadastro de sites personalizados.
- Suporte opcional à API do SteamGridDB.
- Desinstalação seletiva dos atalhos criados pelo pacote.
- Validação formal em Bazzite, SteamOS e outras distribuições.
- Mais serviços de mídia, desde que possam funcionar legalmente por navegador.
- Testes automatizados do parser e gravador de VDF binário.

Itens do roadmap não representam promessa de prazo ou implementação.
