# Steam Media Pack Product Specification

Version: 1.0

Status: Draft

# 1. Product Overview

Steam Media Pack is an Open Source project that transforms Steam into a complete media center for Linux.

It automates the integration of streaming services with Steam by creating launchers, installing artwork and preserving the native Steam experience.

---

# 2. Mission

Transform Steam into the ultimate media center for Linux.

---

# 3. Vision

Become the reference Open Source project for media integration with Steam.

# 7. Product Goals

The project should:

- simplify installation
- automate Steam integration
- preserve user shortcuts
- install artwork automatically
- provide a native Steam experience

---

# 8. Non Goals

The project is NOT intended to:

- replace Steam
- replace browsers
- bypass DRM
- download media
- modify streaming services
