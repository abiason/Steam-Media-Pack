# Código de conduta

Este projeto busca manter um ambiente respeitoso, acolhedor e colaborativo.

## Comportamento esperado

- Tratar as pessoas com respeito.
- Aceitar críticas técnicas construtivas.
- Manter discussões relacionadas ao projeto.
- Proteger dados pessoais e credenciais.

## Comportamento inaceitável

- Assédio, discriminação ou ataques pessoais.
- Publicação de dados privados de terceiros.
- Incentivo à pirataria, fraude ou violação de contas.
- Conteúdo deliberadamente destrutivo ou malicioso.

Problemas de conduta devem ser comunicados aos mantenedores por um canal privado indicado no repositório.
